# FlappyBird
Desenvolvimento de um agente inteligente capaz de aprender a jogar Flappy Bird de forma autônoma. Utilizei Python e o algoritmo NEAT (NeuroEvolution of Augmenting Topologies) para simular o processo de evolução de redes neurais.

Destaques Técnicos:

Implementação de conceitos de Algoritmos Genéticos (População, Mutação, Fitness). 

Manipulação de colisões e física do jogo com a biblioteca Pygame.

Criação de uma rede neural que recebe a distância dos canos e a altura do pássaro como inputs.


O que aprendi: Otimização de hiperparâmetros e lógica de Machine Learning aplicada em tempo real.
